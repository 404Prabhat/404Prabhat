### Self-Hosted System Badge
Deploy a tiny API anywhere (VPS, Raspberry Pi, homelab):

```python
# server.py — FastAPI example
from fastapi import FastAPI
import psutil, subprocess

app = FastAPI()

@app.get("/api/system")
def system_status():
    cpu = psutil.cpu_percent(interval=1)
    mem = psutil.virtual_memory()
    uptime = subprocess.check_output("uptime -p", shell=True).decode().strip()
    return {
        "schemaVersion": 1,
        "label": "system",
        "message": f"{uptime} │ cpu {cpu}% │ mem {mem.used // (1024**3)}G/{mem.total // (1024**3)}G",
        "color": "brightgreen" if cpu < 80 else "yellow",
        "style": "flat-square"
    }
```

```bash
# Run it
pip install fastapi uvicorn psutil
uvicorn server:app --host 0.0.0.0 --port 8080

# Expose via Nginx + Cloudflare Tunnel or your domain
# Then update the badge URL in the README
```

### Auto-Updating README via Cron (optional, max chaos)
```bash
# .github/workflows/update.yml
name: Refresh README
on:
  schedule:
    - cron: '0 */6 * * *'   # every 6 hours
  workflow_dispatch:
jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: python scripts/update_readme.py
      - run: |
          git config user.name "github-actions[bot]"
          git add README.md
          git commit -m "chore: auto-update readme [skip ci]" || exit 0
          git push
```
